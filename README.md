# Tugas Praktikum Aplikasi Berbasis Platform - Aplikasi CRUD Mahasiswa (Node.js)

Aplikasi web sederhana untuk mengelola data mahasiswa menggunakan operasi CRUD penuh (Create, Read, Update, Delete).

## Spesifikasi Teknis
- **Backend Framework**: Node.js (Express)
- **Frontend Framework**: Bootstrap 5
- **Library Javascript**: jQuery
- **Plugin jQuery**: DataTables
- **Format Data**: JSON (AJAX rendering pada DataTables)
- **Struktur Halaman**:
  1. Halaman Home (`index.html`)
  2. Halaman Tabel Data (`data.html`)
  3. Halaman Form Input/Edit (`form.html`)

## Cara Menjalankan Aplikasi
1. Clone repositori ini (https://github.com/mfaittel/ABP_COTS_5.git).
2. Buka terminal di direktori proyek, jalankan `npm install`.
3. Jalankan server dengan perintah `node server.js`.
4. Buka browser pada `http://localhost:3000`.
